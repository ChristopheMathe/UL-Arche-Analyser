# ğŸ§® How Computation Works

1. The user selects or defines the class schedule (one or multiple start/end datetime pairs).

2. The CSV file is opened with the OPEN toolbar action.

3. Optionally, the user filters students.

4. Clicking Compute:
    - The app scans all timestamps in the CSV
    - Keeps only those falling inside the defined schedule ranges
    - Computes the maximum grade for each student during these intervals
    - Displays the results in a new window
